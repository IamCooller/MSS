Firmware for ASIC miners

The Mining Software Solution team has been developing firmware for Antminer ASICs for more than 3 years.
At the moment you can download firmware for Antminer S17e T17E S17 + T17 + S17 S17pro T17 S9 S9i S9j T9 + L3 + L3 ++
All MSS firmware underwent extensive testing before being officially released and became available for download. Compared to stock firmware from Bitmain, MSS firmware has a huge functionality that allows you to manage your ASICs quickly and efficiently and increase the profitability of your equipment up to 50%. More detailed information on all the advantages of the Mining Software Solution firmware you can find on the website.

Advantages

The Mining Software Solution team has been developing firmware for Antminer ASICs for more than 3 years.

Frequency up to 1175Mhz S9, above 850Mhz only for immersion cooling - up to 24TH

Improving energy efficiency by 25% (S9 to 80J / T - 80W / 1TH) and up to 40% (L3 + 1.25W / 1MH)

Fan control, deactivation of control fans for immersion immersion cooling — no more fooling coolers

Selection of paid voltage and frequencies automatically and manually.
Fine tuning frequencies individually

Compatibility with all systems management and monitoring, AP. Parallel Dev fee mining, without drawdowns and suspension of main mining.

Automation for mass firmware S9 series — a patch with all settings and mass update, support for auto-numbering of workers

Mining Firmware for Bitmain ASICs

ATTENTION: BEFORE INSTALLING THE FIRMWARE YOU MUST UNLOCK SSH (use file unlock ssh)!

Unlock SSH is working will all types of the Bitmain firmware except December 2019 If you have Bitmain firmware December 2019 or above you must use SD CARD (you can find it on our web page). After you can flash the asic by MSKminer firmware without unlock. You can check the firmware version in the System - file system version
